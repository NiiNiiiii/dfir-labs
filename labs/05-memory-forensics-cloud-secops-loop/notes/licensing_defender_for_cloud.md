# Licensing note — Defender for Cloud

Defender for Cloud can add posture findings and cloud security signals, but the lab's primary objective was the **Sentinel SecOps loop** built from Sysmon telemetry.

Cost considerations (high level):
- Any Defender plan enablement and additional data ingestion can increase monthly cost.
- Keep cloud security features scoped to the minimum needed for the lab.

This repo intentionally keeps cloud identifiers minimal; see `notes/redactions.md`.
